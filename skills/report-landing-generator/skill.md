# Report Landing Generator (ALTAIR Style)

**Описание:**  
Генерирует эффектный одностраничный HTML-лендинг отчёта Deep Client Analysis строго в фирменном инженерном стиле ALTAIR.

**Обязательные правила стиля (выполняй строго):**
- Тёмная инженерная эстетика: используй тёмные текстуры на фон (из `design/texture/`). Текстура должна растягиваться на всю ширину экрана (`background-size: 100% auto`) и дублироваться по вертикали (`repeat-y`) при длинном контенте. Выбирай текстуры рандомно для каждого нового лендинга.
- Читаемость: тексты, заголовки и футер, попадающие на тёмный фон, должны быть перекрашены (белый/светло-серый) для идеальной читаемости. Карточки могут оставаться светлыми.
- Контейнер: основной блок `div.container` (1320px) обязан иметь фон 50% чёрный + Blur 5px (`backdrop-filter: blur(5px); background: rgba(0,0,0,0.5)`). Это создает "стеклянный" эффект и отделяет контент от фона.
- Эффект гигантских фигур: позади `container`, но над текстурой, создай `fixed` слой (не скроллится). В нём должны быть 3 огромные фигуры (почти во всю высоту экрана, ~120vh) из `design/Figures/`. Они:
  - Выбираются рандомно.
  - Медленно поворачиваются вокруг своей оси.
  - Плавно меняются одна на другую каждые 5 секунд через прозрачность (`opacity`).
  - Позиционируются так, чтобы не пересекаться друг с другом и не заходить более чем на 50% своей ширины под центральный контейнер.
- Скругления: 20–28px для карточек.
- Единственный акцентный цвет — оранжевый (#ED7A1C).
- Шрифты: EuropeExt Bold (uppercase) для заголовков, Montserrat для текста. Подключай EuropeExt через `@font-face` из файла `../../EuropeExt-Bold.otf`.
- Логотип: Обязательно используй официальный файл логотипа `z:\!_STUFF\ALTAIR\assistant-project\design\altair_logo.svg`. В шапке высота логотипа — **минимум 70px**. В HTML вставляй как `img` тег с относительной ссылкой.

**Структура лендинга (обязательная):**
1. Hero: AI-Обзор клиента + название сделки + дата анализа
2. Краткое резюме сделки
3. Портрет клиента (характеристики, сегмент, стиль принятия решений)
4. Ключевые боли и пожелания (с цитатами)
5. Сильные стороны и возможности
6. Авторитетные рекомендации менеджеру
7. Как строить разговор (открывашка, вопросы, аргументы, возражения, close)
8. Следующие шаги
9. Аналитический паспорт (что проанализировано / чего не хватило)
10. Футер с логотипом ALTAIR

**Технические требования:**
- Полноценный standalone HTML5 + Tailwind CDN (или встроенный CSS).
- Responsive (desktop-first).
- Лёгкие fade-up анимации (не перебор).
- В конце страницы — кнопка "Скачать PDF" (placeholder).
- Файл называть: "Deep-Analysis-[CompanyName]-[DealID]-[Date].html"

**После генерации HTML:**
