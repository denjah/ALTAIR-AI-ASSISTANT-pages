# Client Website Researcher

**Описание:**  
Навык глубокого анализа сайта клиента в контексте сделки.

**Инструкции:**
- Для доменов из карточки клиента открывай браузер.
- Анализируй: главную страницу, /about, услуги/продукты, отзывы, новости, контакты.
- Выявляй актуальные боли, тон коммуникации, позиционирование, последние изменения.
- Связывай findings с текущей сделкой.
